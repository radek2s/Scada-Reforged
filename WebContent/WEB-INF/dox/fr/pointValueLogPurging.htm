The Purge now functionality allows the easy purging of point values based on purge strategy. Values that are purged are permanently deleted and unrecoverable following this operation. Select the Purge strategy and press the Purge now button. Deletion is done asynchronously, so you may navigate away from this page or change other point properties, but by doing so you will not receive a purge completed confirmation. The Period allows purging on an age basis. The Values Limit leaves at most as meany values as specified in Values limit field. The All Data deletes all values.
